### Repository Overview

This project contains a dataset and a comprehensive Power BI dashboard focused on the men's apparel market, specifically shirts and t-shirts. The goal is to analyze pricing trends, brand performance, and consumer options in the retail space.

---

Dashboard Link https://app.powerbi.com/links/EF8w-CYK1C?ctid=fa19b9df-6609-4051-89e2-8018d6fb81b4&pbi_source=linkShare

### 1. Dataset Breakdown: `Men+Tshirt.csv`

This CSV file serves as the raw data source for the analysis. It contains retail information for a variety of men's clothing items.

* **Brand**: List of apparel brands including **The Indian Garage Co**, **THE BEAR HOUSE**, **RAYMOND**, **Campus Sutra**, and U.S. Polo Assn..


* **Title**: Detailed descriptions of the items, such as "Men Striped Slim Fit Shirt," "Checked Shirt with Patch Pocket," and "Men Knitted Loose Fit Shirt with Mandarin Collar".


* **Original Price**: The initial retail price of the items (e.g., ₹1,749, ₹2,495).


* **Sale Price**: The discounted price available to consumers (e.g., ₹525, ₹873).


**Use Case**: This data allows for calculating discount percentages, identifying high-value brands, and understanding popular fit styles (Slim Fit, Regular Fit, Loose Fit).

---

### 2. Dashboard Breakdown: `project of cloth.pbix`

The Power BI file provides a visual layer to the CSV data, transforming raw numbers into actionable retail insights.

* **Key Metrics (KPIs)**:
* **Total Brand Reach**: Visualizations showing the market presence of various brands.


* **Pricing Strategy**: Comparison between original and sale prices to highlight consumer savings.




* **Visualizations Included**:
* **Page 1 Layout**: A dedicated report page designed to show itemized sales data.


* **Data Model**: A structured relationship between item categories and pricing tiers.


* **Insights**:
* **Popular Fits**: Trends indicating a high volume of "Slim Fit" and "Checked" shirts in current inventory.


* 
**Brand Performance**: Distribution of products across premium (RAYMOND) and budget-friendly (The Indian Garage Co) segments.





---

### 3. How to Use This Repository

1. **Raw Data**: Use the `Men+Tshirt.csv` for any further Python or SQL analysis.


2. **Visualization**: Open `project of cloth.pbix` in **Power BI Desktop** to interact with the filters, such as filtering by Brand or Fit type, to see how prices shift.


3. **Insights**: Review the dashboard to identify which brands offer the highest discounts or which fit styles are most common in the dataset.
